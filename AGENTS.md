# AGENTS.md

## Cursor Cloud specific instructions

This repository is **documentation-only**. It contains university course study
notes for "Cloud Computing II" and nothing else:

- `README.md` — repo title only.
- `CC-II-Unit-*.txt` — plain-text study notes (the actual content).

There is **no application** here. Specifically, there is:

- No source code (no `.py`, `.js`/`.ts`, `.go`, `.java`, etc.).
- No package manager or dependency manifest (`package.json`, `requirements.txt`,
  `go.mod`, `Cargo.toml`, etc.).
- No build system, test suite, or linter configuration.
- No services, containers, or `.cursor/environment.json`.

As a result there is nothing to install, build, run, lint, or test. The
environment update script is intentionally a no-op. Work in this repo is limited
to editing/adding the Markdown and text notes files.
